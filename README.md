## React Todo MVC

이 템플릿은 react(mobx, redux)를 학습하기 위한 todo list입니다. 기본적으로 css는 구현되어있으며, 실제 이벤트를 바인딩하고 리스닝하는 부분은 직접 구현해야 합니다.

## Getting Started

```bash
yarn install
yarn start
```
